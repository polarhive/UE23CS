# Addressing Modes

| Name               | Form   | Example            | Description |
| ------------------ | ------ | ------------------ | ----------- |
| Immediate          | `$imm` | `movl $-500,%rax`  |             |
| Register           | `ra`   | `movl %rdx,%rax`   |             |
| Direct Addressing  | `imm`  | `movl 2000,%rax`   |             |
| Indirect Adressing | `ra`   | `movl (%rdx),%rax` |             |
